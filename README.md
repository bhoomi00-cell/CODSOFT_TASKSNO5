# CodSoft Python Internship - Task 5: Contact Book

## Description
A simple command-line based Contact Book application built using Python. 
The program allows users to manage their personal contacts by storing 
name, phone number, email, and address, with options to add, view, 
search, update, and delete contacts.

## Features
- Add new contacts with name, phone, email, and address
- View all saved contacts with index numbers
- Search contacts by name or phone number
- Update existing contact details by index
- Delete contacts by index
- Handles invalid input (non-numeric index, out-of-range index)
- Displays a message when the contact list is empty

## Technologies Used
- Python 3

## How to Run
1. Clone this repository
2. Run the file using: `python codsoftTask5.py`
3. Choose an option from the menu (Add/View/Search/Update/Delete/Exit)
4. Follow the prompts to manage your contacts
